Using the magnifier

I have tested the MacOSX Magnifier app on the ESPN Cricinfo website. For zoom in the Command(⌘) key‌ + Plus sign (+) key should be pressed and for zoom out Command(⌘) key‌ + Minus sign (-) key can be used. You can also zoom in and out using the mouse by pressing Command(⌘) key and rotating the wheel on your mouse. The Magnifier is easy to use and has different magnification settings, which is very useful. I have used 200% of magnification. Magnifier is easier to use via its onscreen user interface, which makes it especially user-friendly for mouse users. However, I did find it challenging to move through the screen even to finish reading one sentence; it becomes tiresome. Also, magnification sometimes makes the content blurred though it depneds on how much zoom in has been made. Always custom magnification is the better option for the user because the ratio of magnification depends on the individual ussr which can be shown in the iphone application settings.

Low vision modes

I have included two kinds of low vision modes in this project, magnification mode and contrast alteration mode. At the top of the web page, two distinct buttons have been added to enable each of these modes. The button labelled 'Magnify' makes the fonts on the page larger while the other button labelled 'Change contrast' changes the background and text color.
